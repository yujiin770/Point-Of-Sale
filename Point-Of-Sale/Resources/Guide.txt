Instruction For OOPL  


1. Create a New Project:
    - Open Visual Studio 2010
    - Select "File" -> "New" -> "Project".
    - Choose "Windows Forms Application" under "Visual Basic".
    - Name your project (e.g., "Point-Of-Sale") and select a location.
    - Click "OK".

2.  Add Splash Screen:
    - In the Solution Explorer, right-click on your project.
    - Select "Add" -> "New Item...".
    - Choose "Splash Screen" and name it (e.g., "SplashScreenForm.vb").
    * Click "Add".
    * Design your splash screen:
        - Add a `PictureBox` for your logo or image.
        - Add `Label` controls for your application name ("POINT OF SALES MANAGEMENT"), company name ("Nexus Triad"), and any other relevant information.
        - Add a `ProgressBar` to simulate loading progress (optional).
	- Add button and named it "btnStart"
         Customize the appearance (colors, fonts) as desired.
    	- Configure the Splash Screen:**
        - In the Solution Explorer, double-click "My Project" (or right-click the project and select "Properties").
        - Go to the "Application" tab.
        - In the "Splash screen" dropdown, select "SplashScreenForm" (or the name you gave it).
   * **Set the Minimum Display Time:**
            - In the `SplashScreen.vb` code, add a `Timer` control to the form.
            - Set the `Interval` property of the `Timer` to the desired display time in milliseconds (ex., Interval: 3000 for 3 seconds).
            - In the `Timer_Tick` event handler, add code to close the splash screen and open the main login form:
